## Задание

Задание размещено в папке `task` в файле `README.md`.

В рамках проектной (учебной) практики был разработан прототип Telegram-бота для изучения русского языка.

Основная цель проекта — реализация интерактивной обучающей системы, включающей:
- предоставление учебных материалов по русскому языку;
- систему упражнений и тестов;
- обработку пользовательского ввода;
- отслеживание прогресса пользователя;
- реализацию логики диалога с пользователем.

Бот разработан с использованием языка программирования Python и предназначен для удобного и доступного изучения русского языка в формате общения.

---

## Вариативная часть задания

В рамках вариативной части проекта была реализована система геймификации процесса обучения.

Геймификация является важным инструментом повышения вовлечённости пользователей, так как позволяет сделать обучение более интересным и мотивирующим. В связи с этим в проект была добавлена мини-игра в жанре roguelike, разработанная на C#.

Игровой модуль включает:
- пошаговую систему перемещения;
- взаимодействие с противниками;
- базовую боевую систему;
- использование способностей персонажа.

Добавление игрового элемента позволяет разнообразить процесс обучения, повысить интерес пользователя и закреплять знания в интерактивной форме.
