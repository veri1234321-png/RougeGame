## Задание

Задание размещено в папке `task` в файле `README.md`.

В рамках проектной (учебной) практики был разработан прототип пошаговой roguelike-игры на основе библиотеки **RogueSharp** и графической оболочки **RLNET**.

Основная цель проекта — реализация игровой логики подземелья, включая:
- генерацию уровней;
- систему перемещения персонажа;
- боевую систему;
- взаимодействие с монстрами;
- систему способностей персонажа.


Также была расширена система поведения монстров, включая поддержку состояний (например, заморозка), влияющих на их ход.

---

## Вариативная часть задания

В рамках вариативной части проекта была реализована система магических способностей с различными эффектами воздействия на противников.

Дополнительно была добавлена контрольная способность (Freeze), позволяющая:
- временно блокировать действия противников;
- управлять тактическим поведением врагов;
- расширить стратегические возможности игрока.

Источники и материалы:
- учебный курс по разработке roguelike-игр на основе RogueSharp;
- документация библиотеки RogueSharp;
- практические примеры реализации боевых систем.
